# Minecraft Single-File Edition 🧱

A lightweight, high-performance Minecraft clone built entirely within a single HTML file. It uses raw WebGL and procedural generation to create infinite 3D worlds with zero external dependencies.

## 🚀 Key Highlights
* **All-in-One:** Entire game logic, rendering, and assets in one file.
* **Zero Dependencies:** Runs natively in any modern browser.
* **Procedural:** Infinite, unique landscapes every time you play.
* **Persistent:** Saves your world directly to your browser's local storage.

## 🕹️ Controls
* **W A S D:** Move.
* **Mouse:** Look around.
* **Space:** Jump / Swim.
* **Left Click:** Break blocks.
* **Right Click:** Place blocks.
* **E:** Inventory.
* **G:** Debug info.

## 🛠️ Technical Stack
* **WebGL:** For 3D rendering.
* **Procedural Texture Generation:** No external image files required.
* **Voxel Engine:** A custom, performant block-based building system.
